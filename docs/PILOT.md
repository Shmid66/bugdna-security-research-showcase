# Bounded Solidity Pre-audit Pilot

A small human-led, tool-assisted review of an exact authorized Solidity scope. BugDNA accelerates intake and hypothesis ranking; the analyst remains responsible for reachability, impact, false-positive rejection, remediation, and any permitted local proof.

## Proposed pilot shape

- Agree written authorization, exact file scope, rules, deadline, and compensation route before work starts
- Create a hash-locked scope manifest and stable run identifier
- Run the local scanner without exporting full target source
- Manually accept or reject every material hypothesis
- Deliver a concise root-cause and remediation report with an explicit coverage boundary

## Explicit exclusions

- No autonomous-audit or complete-coverage claim
- No live exploitation, fund movement, or out-of-scope probing
- No identity, residency, KYC, sanctions, or payment circumvention
- No severity or exploitability claim without human validation
- No automatic bounty or contest submission

## Start condition

Work starts only after written authorization, exact scope, permitted local proof method, delivery format, eligibility, and compensation route are confirmed.
A scanner hypothesis is not presented as a vulnerability until a human analyst validates root cause, reachability, impact, and false-positive conditions.
