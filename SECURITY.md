# Security and Responsible Use

This showcase does not authorize testing of any system.

Use BugDNA only on local educational targets, public historical fixtures, or source for which exact written authorization and scope are confirmed. Do not test mainnet or public testnet deployments unless the applicable rules explicitly permit it. Do not move funds, evade identity or residency controls, collect credentials, or publish an unpatched vulnerability.

The public pack contains no vulnerability submission channel. A future disclosure workflow must follow the exact rules of the authorized project or platform.
